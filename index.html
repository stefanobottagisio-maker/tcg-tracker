La mia collezione

Fonti dati e immagini: One Piece via CDN Limitless TCG (fallback siti Bandai); Yu-Gi-Oh! via API YGOPRODeck (che include il prezzo Cardmarket in euro); Lorcana via API Lorcast. Serve connessione internet per caricare immagini e set non ancora visitati; gli elenchi carte di Yu-Gi-Oh! e Lorcana vengono salvati in locale dopo la prima visita. La collezione è salvata nella memoria del browser: usa “Esporta backup” regolarmente. Per Yu-Gi-Oh! il conteggio è per ristampa nel set visitato (stessa carta in due set = due voci). Scanner: il pulsante “Scansiona carta” apre la fotocamera dal vivo – inquadra il codice stampato nel riquadro e la carta viene riconosciuta e aggiunta da sola (One Piece: angolo in basso a sinistra, es. OP05-119; Yu-Gi-Oh!: codice sotto l'illustrazione, es. MRD-EN060; Lorcana: riga in basso, es. 12/204 EN 3). In alternativa “Usa una foto” per uno scatto singolo. Il motore OCR viene scaricato al primo utilizzo; la fotocamera dal vivo richiede una connessione HTTPS (come GitHub Pages).
